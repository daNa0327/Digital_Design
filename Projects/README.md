# Digital Design Projects
Verilog HDL을 활용하여 설계한 디지털 회로 프로젝트 저장소입니다.

각 프로젝트는 RTL 설계, Testbench 작성, 시뮬레이션 및 파형 분석, 결과 검증까지의 과정을 포함합니다.
