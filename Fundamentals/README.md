# Digital Design Fundamentals
기초적인 Verilog HDL 학습을 위한 논리회로 구현 모듈 저장소입니다.

각 회로는 RTL 설계, Testbench 작성, 시뮬레이션, 그리고 직접 작성한 진리표, 논리식, 회로도를 포함하여 학습 내용을 정리합니다.
